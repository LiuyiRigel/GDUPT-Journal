## 📝 使用指南

### 编辑内容

```latex
% 修改标题（第60行）
{\zihao{2-} \heiti 你的标题 }

% 修改作者（第64行）
{\zihao{4-}\kaishu 作者名$^{1}$\quad 作者名$^{2}$ }

% 修改摘要（第75行）
\noindent {\heiti 摘要: }{\fangsong
你的摘要内容}

% 修改关键词（第81行）
{\heiti 关键词：}{\fangsong
你的关键词}
```

### 添加图片

```latex
\begin{figure}[htbp]
\centering
\includegraphics[width=0.6\linewidth]{your_image.pdf}
\caption{图片说明}
\label{fig2}
\end{figure}
```

### 添加表格

```latex
\begin{table}[htbp]
\centering
\songti 表X\quad 表说明
\begin{tabular}{ll}
\toprule
标题1 & 标题2 \\
\midrule
内容1 & 内容2 \\
\bottomrule
\end{tabular}
\label{tab2}
\end{table}
```
